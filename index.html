PHP Programming Test

To apply for the programming position available at Inetasia please:

  • Do both Test 1 & Test 2 below
  • Put both tests in a folder marked with your name and zip it up
  • Email your test, CV, and Cover letter to HR@inetasia.com

We care mostly that things are done correctly so please spend some extra time to check what you have done before sending it to us.

Test 1

Recreate the following buttons in HTML and CSS as closely as possible:

http://inetasia.github.io/score-buttons.png

For each button use only a single <a> tag, with as many <span> tags as you need.

Test 2

In this test your code and implementation will be evaluated. Please try to program as you normally would when maintaining a piece of software. During this test you may use the internet, stackoverflow, and any other resource you normally use during development with the exception of asking friends for help during this test.

Please do not use large frameworks such as laravel or codeigniter for this test. You may use composer packages however please write most of the code yourself.

Please create a program that when run creates a unique list of coupon codes.

It must:

  • create 250 codes
  • the codes must be 10 characters long
  • the codes must begin with "IA"
  • write the 250 codes to a file

When you have finished this task please generate the 250 codes into a file called "test2".
